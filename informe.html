Variables Implementadas

Modelo COCOMO 81 Intermedio

Variables Básicas:
  • Tamaño del proyecto (KLOC)
  • Modo de desarrollo (Orgánico)
  • Salario promedio mensual
Cálculos Alternativos:
  • Tamaño del equipo para calcular duración
  • Duración deseada para calcular el equipo necesario
Factores de Ajuste Implementados:
  • RELY - Fiabilidad requerida del software
  • AEXP - Experiencia en la aplicación
  • TOOL - Uso de herramientas software

Ejemplo de Cálculo

Datos de entrada:
  • Tamaño: 25 KLOC (25,000 líneas de código)
  • Modo: Orgánico
  • Factores de ajuste: EAF = 1.15
  • Salario: 4,000,000 COP mensuales
Cálculos:
  • Esfuerzo = 2.4 × (25)1.05 × 1.15 = 80.71 persona-meses
  • Duración = 2.5 × (80.71)0.38 = 10.64 meses
  • Personal promedio = 80.71 / 10.64 = 7.59 personas
Interpretación:

Este proyecto requeriría aproximadamente 81 persona-meses de esfuerzo, lo que significa un equipo de 7-8 personas trabajando durante 10 a 11 meses. El costo total estimado, considerando un salario mensual de 4,000,000 COP, sería de 322,840,000 COP.
